CompTIA A+ 220-1101 - Flash Memory Cards

Memory Card Types

Memory cards are removable flash storage devices used in:

  • Digital cameras
  • Smartphones (Android)
  • Tablets
  • Action cameras (GoPro, etc.)
  • Drones
  • Gaming devices (Nintendo Switch)

SD (Secure Digital) Card

Physical Size: 32mm Ã— 24mm Ã— 2.1mm

Common Use: Digital cameras, camcorders

SD Card Generations:

  • SD: Up to 2 GB
  • SDHC (High Capacity): 4 GB to 32 GB
  • SDXC (Extended Capacity): 64 GB to 2 TB
  • SDUC (Ultra Capacity): 2 TB to 128 TB (newer, rare)

Features:

  • Write-protect switch on side
  • Most versatile - works in many devices
  • Different speed classes for different uses

MicroSD Card

Physical Size: 15mm Ã— 11mm Ã— 1mm (much smaller than SD)

Common Use: Smartphones, tablets, action cameras, drones

MicroSD Generations (same as SD):

  • MicroSD: Up to 2 GB
  • MicroSDHC: 4 GB to 32 GB
  • MicroSDXC: 64 GB to 2 TB

Features:

  • Can be used in SD slot with adapter
  • Most common in mobile devices
  • Same speed classes as full-size SD
  • NOT compatible with iOS devices (iPhones/iPads)

CompactFlash (CF)

Physical Size: 43mm Ã— 36mm Ã— 3.3mm (larger, thicker)

Common Use: Professional cameras, older devices

Features:

  • Older technology
  • Still used in some professional cameras
  • Very durable
  • Higher cost per GB
  • Being replaced by faster card types

Types:

  • CFast: Faster version using SATA interface
  • CFexpress: Newest, uses PCIe/NVMe (very fast)

Application Performance Classes

Designed for running apps from SD card (Android "Adoptable Storage"):

  • A1: Minimum 1500 read IOPS, 500 write IOPS - Good for apps
  • A2: Minimum 4000 read IOPS, 2000 write IOPS - Better for apps

Compatibility & Formatting

Device Compatibility

  • Backward compatible: SDXC reader can read SDHC and SD cards
  • NOT forward compatible: SD reader cannot read SDXC cards
  • Speed class: Device must support the speed rating to get full speed
  • MicroSD adapter: Allows microSD to work in SD slot

File Systems

Card Type Default File System Why
SD (up to 2GB) FAT16 Older standard
SDHC (4-32GB) FAT32 Universal compatibility
SDXC (64GB+) exFAT Supports large files (>4GB)

Common Issues & Solutions

Problem Cause Solution
Card not recognized Dirty contacts, incompatible format, dead card Clean contacts, try different device, reformat (loses data)
Cannot write to card Write-protect switch enabled (SD cards) Slide switch on side of card to unlock position
Card full but files deleted Hidden files, corrupted filesystem Show hidden files, format card
Slow write speed Low speed class card, fragmentation Use faster speed class card (U3, V30+)
Video recording stops Card too slow for video bitrate Use U3 or V30+ rated card for 4K video
Files corrupted Removed while writing, fake card Always safely eject, buy from reputable sellers

Important Notes

iOS Devices (iPhones/iPads)

  • Do NOT support SD or microSD cards
  • No expandable storage option
  • Must buy device with desired storage capacity

Fake/Counterfeit Cards

  • Common problem with memory cards
  • Advertised capacity higher than actual
  • May report as full size but corrupt data
  • Buy from reputable sellers (Amazon, Best Buy, B&H)
  • Test with H2testw (Windows) or F3 (Mac/Linux)

Best Practices

  • Always safely eject before removing
  • Format in the device that will use it
  • Don't fill card 100% - leave some free space
  • Keep contacts clean
  • Store in protective case when not in use
  • Backup important data - cards can fail
