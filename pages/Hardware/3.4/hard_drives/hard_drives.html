What is a Hard Drive (HDD)?

A **Hard Disk Drive (HDD)** is a storage device with spinning magnetic platters that store data. It has moving parts (platters spin on a spindle, read/write heads move). It's slower than SSDs but cheaper for large storage (TB capacities).

Spindle speed = how fast the platters spin. Faster RPM = quicker data access (lower latency), but more heat, noise, power use, and usually less capacity for the price.

Form Factors (Physical Sizes)

Form factor = physical size/shape of the drive. Affects where it fits (desktop vs laptop). Both use same SATA connectors usually.

  • 3.5-inch â†’ Larger, for desktops and servers. More platters = higher capacity (e.g., 8-20+ TB easy). Needs more power/space.
  • 2.5-inch â†’ Smaller/thinner, for laptops, small desktops, external drives. Lower capacity max, but fits tight spaces. Can use in desktop with adapter/bracket.
Exam Tip: Know only 2.5" and 3.5" for HDDs. 2.5" = laptops/portables, 3.5" = desktops. (SSDs use these too, plus M.2.)

Quick Exam Cheat Sheet

  • HDD = spinning magnetic platters, moving parts, cheap large storage
  • Spindle speeds (RPM): 5400 (laptops/basic), 7200 (standard desktop), 10,000/15,000 (fast/enterprise, hotter)
  • Form factors: 3.5-inch (desktops/servers, bigger capacity), 2.5-inch (laptops/portables, fits small spaces)
  • Higher RPM = faster access but more power/noise/heat
  • Connect via SATA (most common), same for both sizes
