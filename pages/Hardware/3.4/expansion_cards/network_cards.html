What is a NIC Card?

A **NIC** (Network Interface Card) connects your computer to a network (usually wired Ethernet). Every device on a network needs one to send/receive data.

  • Has a unique **MAC address** (hardware address) burned in
  • Handles converting data to signals for the cable
  • Found in PCs, laptops, servers, printers â€” everywhere networked
Exam Tip: NICs are expansion cards in Domain 3.0 Hardware (same as sound/video/capture cards). Also mentioned in Domain 2.0 Networking hardware.

Two Main Types

Integrated / Onboard NIC

  • Built into the motherboard (most common today)
  • Port on the back I/O panel
  • Good for normal use: home/office, internet, file sharing
  • Common speeds: 1 Gbps (Gigabit), now often 2.5 Gbps or 5 Gbps

Discrete / Expansion NIC

  • Separate card plugged into PCIe slot (usually x1)
  • Used when: onboard fails, need faster speed, more ports, or server/multi-NIC setup
  • Can have 1, 2, 4+ ports (multi-port for servers)
  • Supports higher speeds like 10 Gbps in some cases

Common Port / Connector

  • RJ45 â†’ The standard Ethernet port (looks like big phone jack, 8 pins)
  • Used with twisted-pair cables (Cat5e, Cat6, etc.)
  • LEDs on port: Green/amber lights show link and activity/speed
Exam loves this: RJ45 is the main wired NIC port. Know it's for Ethernet (not phone RJ11). LEDs help troubleshoot connection.

Installing a NIC Card (Simple Steps)

  • Power off PC, unplug.
  • Open case, insert into PCIe x1 slot until it clicks.
  • Secure bracket to case.
  • Close case, plug Ethernet cable into **new NIC port**.
  • Boot up â†’ install drivers (from manufacturer or Windows usually auto-detects).

Troubleshooting NIC Issues

  • No connection â†’ Check cable/link lights, wrong port used, drivers missing/bad, cable damaged.
  • Slow speed â†’ Wrong cable (e.g., old Cat5 instead of Cat6), duplex/speed mismatch, faulty NIC.
  • No link light â†’ Bad cable, port issue, power problem, or NIC dead.
  • After install no network â†’ Plugged into old onboard port instead of new card, disable onboard in BIOS if conflicting.
