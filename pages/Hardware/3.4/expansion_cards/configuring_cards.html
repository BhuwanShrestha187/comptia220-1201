What are Device Drivers?

Drivers are software that let Windows talk to hardware (like video cards, sound cards, NICs). Most devices work with basic drivers, but for full features/performance, install manufacturer drivers.

Configuring Devices (Properties & Manufacturer Tools)

Most devices are preconfigured, but check:

  • Device Manager: Right-click device â†’ Properties â†’ Driver tab (update/roll back), Details, Events, or advanced tabs (vary by device).
  • Video cards: Set resolution, refresh rate, multiple monitors (via NVIDIA/AMD control panel app).
  • Sound cards: Usually no config needed; advanced features via OS or manufacturer app.
  • Video capture cards: Often need special software/utilities (not native to Windows) for recording/streaming settings.
  • Wireless NICs: Install screw-on antenna AFTER securing card; configure SSID/security via Windows or manufacturer tool (can be tricky for beginners).
  • Wired NICs: Easier â€“ configure IP, speed/duplex in adapter properties if needed (advanced networking topics later).

Installing Expansion Cards (General Steps)

  1. Power off PC, unplug, ground yourself (anti-static wrist strap).
  2. Open case, install card in open PCIe slot (match type/size).
  3. Connect extra power if needed (e.g., video cards â€“ 6/8-pin).
  4. Close case, boot up.
  5. Install driver (PnP may auto-do; otherwise from disc/site).
  6. Check BIOS/UEFI if not recognized (enable slot, resources).
  7. Open Device Manager: Look for device, no yellow ! error badge.
  8. Use manufacturer utility for extra config if provided.
  9. Always check docs/website for special steps.
