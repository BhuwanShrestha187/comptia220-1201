What is SATA?

Serial ATA (SATA) is the standard interface for connecting storage devices like SSDs, HDDs, and Optical Drives to the motherboard. It replaced the older, bulky PATA (IDE) cables.

SATA Revisions & Speeds

Exam Tip: You must memorize the transfer rates (Gb/s) for each revision.

Revision Speed (Gb/s) Max Throughput (approx) Status
SATA 1.0 1.5 Gb/s 150 MB/s Obsolete
SATA 2.0 3.0 Gb/s 300 MB/s Legacy / Older PCs
SATA 3.0 6.0 Gb/s 600 MB/s Current Standard
SATA 3.2 16 Gb/s 1900 MB/s SATA Express (Rarely used)

Cables & Connectors

1. Data Cable (7-Pin)

  • Pins: 7 pins total.
  • Shape: L-shaped key connector (prevents plugging it in upside down).
  • Length: Maximum cable length is 1 meter (3.3 feet).
  • Topology: Point-to-point (one cable = one drive). No "Master/Slave" jumpers like PATA.

2. Power Connector (15-Pin)

  • Pins: 15 pins (wider than the data cable).
  • Voltages: Provides 3.3V, 5V, and 12V power rails.
  • Purpose: Replaced the old 4-pin Molex power connector.

3. eSATA (External SATA)

  • Usage: For connecting external external hard drives.
  • Connector: Different shape (flat/rectangular) than internal SATA to prevent mistakes.
  • Length: Max length 2 meters.
  • Power: Standard eSATA does not provide power (requires separate power adapter), unlike USB.

SATA vs. NVMe (M.2 Form Factor)

Feature M.2 SATA SSD M.2 NVMe SSD
Interface Uses SATA bus (max 600 MB/s) Uses PCIe bus (up to 7,000+ MB/s)
Keying Usually B+M Key (two notches) Usually M Key (one notch on right)
Compatibility Works in M.2 slots that support SATA Works in M.2 slots that support PCIe

Configuration: AHCI vs. RAID

AHCI (Advanced Host Controller Interface)

  • The standard mode SATA drives should run in.
  • Enables NCQ (Native Command Queuing): Optimizes the order in which the drive executes read/write commands.
  • Enables Hot Swapping: Ability to plug/unplug the drive while the PC is on (if the port supports it).

ðŸŽ¯ Key Points for the Exam

  • SATA 1.0: 1.5 Gb/s
  • SATA 2.0: 3.0 Gb/s
  • SATA 3.0: 6.0 Gb/s (The modern standard)
  • Data Connector: 7 pins, L-shaped.
  • Power Connector: 15 pins, L-shaped, 3.3V/5V/12V.
  • Max Length: 1 meter (Internal), 2 meters (eSATA).
  • AHCI: The mode required for Hot-swapping and NCQ.
  • 1-to-1: One device per SATA port (no daisy chaining).
  • M.2 SATA: Fits M.2 slot but runs at SATA 3 speeds (slower than NVMe).
  • Hot Swappable: Yes, if AHCI is enabled.
