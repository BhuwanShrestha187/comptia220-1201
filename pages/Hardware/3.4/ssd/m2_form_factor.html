What is M.2?

M.2 (formerly Next Generation Form Factor - NGFF) is a specification for internally mounted computer expansion cards and connectors. It replaces the older mSATA standard and is widely used for high-performance SSDs, Wi-Fi cards, and other devices. M.2 SSDs connect directly to the motherboard without cables, saving space and enabling faster speeds.

M.2 Key Types & Sockets

Key Notch Position Common Interfaces Typical Use
B key Notch on left (pins 12-19) SATA, PCIe x2 Lower-cost SSDs, WWAN, Wi-Fi (legacy)
M key Notch on right (pins 59-66) PCIe x4, NVMe, SATA (some) High-performance NVMe SSDs
B+M key Both notches SATA or PCIe x2 Compatibility with both sockets (usually SATA)

Important: The key must match the motherboard slot. M-key SSDs only fit M-key slots; B+M key SSDs fit either but may run at slower speeds.

M.2 Lengths (Form Factors)

M.2 drives are identified by their width (always 22mm) and length. Common lengths:

  • 2230: 22mm wide, 30mm long (often for Wi-Fi cards or small embedded)
  • 2242: 22mm x 42mm (some older SSDs)
  • 2260: 22mm x 60mm (less common)
  • 2280: 22mm x 80mm (most common for desktops/laptops)
  • 22110: 22mm x 110mm (enterprise/server SSDs)

Motherboards have mounting points for various lengths. Always check supported lengths before buying.

Key Advantages of M.2

  • Small footprint: Saves space for thinner laptops and compact desktops.
  • High performance: NVMe M.2 drives dramatically outperform SATA SSDs.
  • No cables: Direct motherboard connection reduces clutter and improves airflow.
  • Multi-function: Same slot can support SSDs, Wi-Fi, Bluetooth, or WWAN cards.
  • Lower latency: NVMe protocol is optimized for SSDs with parallel I/O.

Motherboard Compatibility

Checklist before buying M.2 SSD:

  • Key type: Does the slot accept M-key, B-key, or both?
  • Interface support: Does the slot support NVMe (PCIe) or only SATA? (Many older slots are SATA-only.)
  • PCIe generation: PCIe 3.0, 4.0, or 5.0? Backward compatible but slower.
  • Physical length: Does your board have standoffs for 2280? (most do)
  • BIOS/UEFI: NVMe drives may need UEFI and sometimes a BIOS update for boot.
  • Shared lanes: Some M.2 slots share bandwidth with SATA ports or PCIe slots â€“ check motherboard manual.

Installation Tips

Step Detail
1. Locate M.2 slot Usually near CPU or chipset; may have a heatsink (especially PCIe 4.0/5.0).
2. Remove standoff Move the small screw/standoff to match your drive length (e.g., 2280).
3. Insert at angle Insert drive at ~30-degree angle, gold fingers first, then press down.
4. Secure with screw Use the tiny screw provided with motherboard (don't overtighten).
5. Attach heatsink (if any) High-end NVMe drives generate heat; some boards include thermal pads.

Troubleshooting M.2 SSDs

Issue Possible Cause Solution
Drive not detected in BIOS Incompatible key, SATA mode vs NVMe, not seated fully Reseat drive; check BIOS settings (enable NVMe or disable CSM); verify key compatibility
Drive detected but slow Running in SATA mode, PCIe lane limitations, old driver Check if slot is PCIe or SATA; update NVMe driver; verify PCIe link speed
Overheating/throttling Lack of airflow or heatsink Add M.2 heatsink; ensure case airflow; monitor temps with tools
Cannot boot from drive BIOS not set to UEFI, partition table (MBR vs GPT) Switch to UEFI boot; use GPT partition; enable NVMe in boot order
Drive drops out randomly Power management settings, loose connection Reseat drive; update BIOS; disable PCIe link state power management

M.2 Beyond SSDs

  • Wi-Fi / Bluetooth cards: Usually key E or A+E, smaller size (2230).
  • WWAN / Cellular modems: Often B-key or E-key.
  • NVMe to PCIe adapters: Some M.2 slots can be adapted for other devices.
  • Intel Optane Memory: Used as cache (requires specific chipset).

Important A+ Exam Tips

  • Know the difference between SATA M.2 and NVMe M.2 â€“ they are physically similar but use different protocols.
  • Recognize that M key is typically for NVMe (PCIe x4), while B key is often SATA or PCIe x2.
  • Remember common length: 2280 is the most popular for SSDs.
  • Understand that M.2 slots can support multiple functions, but you must check motherboard specs.
  • NVMe = Non-Volatile Memory Express (optimized for SSDs, lower latency).
  • Older motherboards may have M.2 slots that are SATA only â€“ NVMe drives won't work.
