Keyboard Types

Wired Keyboards

  • USB: Most common connection type
  • PS/2: Legacy purple connector (rare on modern systems)
  • More reliable than wireless
  • No battery needed
  • Preferred for BIOS/firmware updates

Wireless Keyboards

  • Bluetooth: Built into most laptops/tablets
  • RF (Radio Frequency): USB dongle receiver
  • Requires batteries or charging
  • Can have connectivity issues
  • Won't work in BIOS (most systems)

Mobile Device Keyboards

Virtual/On-Screen Keyboards

Standard on smartphones and tablets. No physical keys. Types on touchscreen.

  • Autocorrect and predictive text features
  • Swipe typing capability
  • Multiple language support
  • Can be hidden to maximize screen space

Physical Mobile Keyboards

Bluetooth Keyboards

  • External keyboard connects wirelessly
  • Must be paired with device
  • Requires battery or charging
  • Full-size typing experience

Laptop Keyboards

  • Built into the device
  • Smaller keys than desktop
  • Fn (Function) key for special features
  • Often lacks numeric keypad

Special Keys & Function Keys

Key/Key Combo Function
Fn (Function) Used with other keys for special functions (brightness, volume, etc.)
Fn + F keys Screen brightness, volume, Wi-Fi toggle, airplane mode, keyboard backlight
Fn + Arrow Keys Often controls Home, End, Page Up, Page Down
Ctrl + Alt + Del Windows security screen, Task Manager access
Windows Key Opens Start menu, used in many shortcuts
Num Lock Toggles numeric keypad on/off
Caps Lock Toggles all caps typing
Scroll Lock Legacy key, rarely used today

Laptop Keyboard Issues

Keys Not Working

Common Causes:

  • Debris under keys (food, dust, liquid)
  • Damaged keyboard cable inside laptop
  • Driver issues (update keyboard drivers)
  • Physical damage to keyboard
  • Num Lock accidentally enabled (on laptops without numeric keypad)

Solutions:

  • Clean keyboard with compressed air
  • Test with external USB keyboard to isolate problem
  • Check if Fn lock or Num Lock is activated
  • Update keyboard drivers in Device Manager
  • Replace keyboard if hardware failure

Sticky Keys

Physical Sticky: Keys stick when pressed due to liquid spill or debris

  • Power off device immediately if liquid spilled
  • Clean with compressed air and isopropyl alcohol
  • May require key cap removal or keyboard replacement

Sticky Keys Feature: Windows accessibility feature

  • Activated by pressing Shift 5 times
  • Allows one-key-at-a-time pressing of key combos
  • Can be disabled in Accessibility settings

Laptop Keyboard Replacement

  • Usually requires partial disassembly of laptop
  • Keyboard connects via ribbon cable to motherboard
  • Must match exact model for proper fit
  • Some laptops have keyboards riveted (very difficult to replace)
  • External USB keyboard is quick workaround

Keyboard Connectivity Issues

Problem Solution
Wireless keyboard not responding Check batteries, re-pair Bluetooth, check USB dongle connection
Keyboard not recognized in BIOS Use wired USB keyboard (wireless often doesn't work in BIOS)
Some keys type wrong characters Check keyboard language settings in OS
Keyboard works but trackpad doesn't Fn key may have disabled trackpad (check Fn + touchpad key)
Bluetooth keyboard won't pair Put keyboard in pairing mode, remove old pairing, check Bluetooth is on

Mobile Device Keyboard Features

Touchscreen Keyboard Settings

  • Haptic feedback (vibration)
  • Key press sounds
  • Autocorrect on/off
  • Predictive text
  • Swipe/gesture typing
  • Multiple language keyboards

Common Issues

  • Keyboard won't appear: Restart app or device
  • Wrong language: Change input method in settings
  • Autocorrect errors: Adjust or disable in settings
  • Slow response: Clear cache or restart device

1. A user reports their laptop keyboard is not working, but an external USB keyboard works fine. What is the MOST likely cause?
Correct! If an external keyboard works but the internal doesn't, the problem is isolated to the laptop keyboard itself or its connection (ribbon cable) to the motherboard. The USB controller works (proven by external keyboard), so it's a hardware issue specific to the internal keyboard.
2. Which keyboard connection type will NOT work when accessing BIOS/UEFI on most systems?
Correct! Bluetooth keyboards require the operating system to be loaded for the Bluetooth drivers to work. In BIOS/UEFI (pre-boot environment), these drivers aren't loaded, so Bluetooth keyboards won't function. Use wired USB or PS/2 keyboards for BIOS access.
3. A user accidentally pressed Shift 5 times and now modifier keys behave strangely. What feature was activated?
Correct! Sticky Keys is a Windows accessibility feature activated by pressing Shift 5 times rapidly. It allows users to press key combinations one key at a time instead of simultaneously. You can disable it in Accessibility settings or by pressing Shift 5 times again.
4. On a laptop, what key is typically used in combination with function keys to adjust screen brightness or volume?
Correct! The Fn (Function) key is used on laptops to activate secondary functions of keys, including adjusting brightness, volume, Wi-Fi, airplane mode, and keyboard backlighting. It's typically pressed in combination with F keys or other special keys.
5. A technician spills coffee on a laptop keyboard. What should be done FIRST?
Correct! Power off immediately to prevent electrical shorts and component damage. Then disconnect power and remove the battery if possible. Liquid damage can cause shorts that destroy components if power is applied. After powering off, clean and dry thoroughly before attempting to use again.
6. A user's wireless keyboard suddenly stops working. What should be checked FIRST?
Correct! Batteries are the most common failure point for wireless keyboards and the easiest to check. Dead or low batteries are the #1 cause of wireless keyboard failure. Always start with the simplest and most likely solution before moving to more complex troubleshooting.
7. What type of keyboard is standard on smartphones and tablets?
Correct! Virtual or on-screen keyboards are standard on modern touchscreen devices like smartphones and tablets. They appear on the screen when text input is needed and can be hidden when not in use. They support features like autocorrect, predictive text, and swipe typing.
8. Some keys on a laptop are typing the wrong characters. What should be checked?
Correct! When keys type incorrect characters, the keyboard language or region settings are likely set incorrectly. Different keyboard layouts (US, UK, International) have different character mappings. Check and change the input language in the operating system settings.

  • Virtual/on-screen keyboards are standard on smartphones and tablets
  • Bluetooth keyboards must be paired with mobile devices and require charging/batteries
  • Sticky Keys = Windows accessibility feature activated by pressing Shift 5 times

  • Wireless keyboard not working: Check batteries FIRST (most common cause)
  • Keys typing wrong characters: Check keyboard language/region settings
  • Laptop keyboard replacement: Requires disassembly, ribbon cable connection, model-specific part
  • External USB keyboard is quick troubleshooting tool and workaround for failed laptop keyboard
  • Num Lock can cause issues on laptops without numeric keypads
  • PS/2 connector: Purple (keyboard) or green (mouse), legacy technology
