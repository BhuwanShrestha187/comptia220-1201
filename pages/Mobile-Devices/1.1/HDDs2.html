Laptop Storage Types

2.5" HDD

  • Traditional spinning hard drive
  • Slower (50-120 MB/s)
  • More fragile (moving parts)
  • Cheaper per GB
  • Common: 500GB, 1TB, 2TB
  • 7mm or 9.5mm thick
  • SATA interface

2.5" SATA SSD

  • Solid state, no moving parts
  • Much faster (up to 550 MB/s)
  • More durable
  • More expensive per GB
  • Common: 128GB, 256GB, 512GB, 1TB
  • 7mm thick (standard)
  • SATA interface
  • Direct replacement for 2.5" HDD

M.2 SSD

  • Small stick form factor
  • Plugs directly into motherboard
  • SATA or NVMe protocol
  • NVMe much faster (3500+ MB/s)
  • Common sizes: 2242, 2260, 2280
  • 2280 = 22mm wide, 80mm long
  • Saves space in thin laptops

1.8" Drive (Legacy)

  • Used in older ultrabooks
  • Smaller than 2.5"
  • Less common today
  • Replaced by M.2 drives

M.2 Drives - Critical Details

M.2 Form Factor Sizes

  • 2242: 22mm x 42mm
  • 2260: 22mm x 60mm
  • 2280: 22mm x 80mm (most common in laptops)
  • 22110: 22mm x 110mm (rare, some desktops)

Note: Width is always 22mm. Length varies. Check laptop spec for compatible size.

M.2 Key Types (Connector Notches)

  • B Key: SATA SSDs, PCIe x2
  • M Key: NVMe SSDs, PCIe x4 (faster)
  • B+M Key: Both notches, works in both slots

Important: The notch position determines what slots the drive fits. Must match the motherboard slot.

M.2 SATA

  • Uses SATA protocol
  • Same speed as 2.5" SATA SSD (~550 MB/s)
  • B+M key usually
  • Cheaper than NVMe

M.2 NVMe

  • Uses NVMe protocol over PCIe
  • Much faster (1500-3500+ MB/s)
  • M key usually
  • More expensive
  • Best for performance

Laptop Drive Replacement

Replacing 2.5" Drive (HDD or SSD)

  1. Power off laptop completely
  2. Remove battery if removable
  3. Remove bottom panel or drive bay cover
  4. Remove screws holding drive caddy/bracket
  5. Slide out old drive
  6. Remove mounting screws from old drive
  7. Mount screws on new drive
  8. Slide new drive into bay
  9. Connect SATA connector
  10. Secure and reassemble

Installing M.2 SSD

  1. Power off and disconnect power
  2. Remove bottom panel
  3. Locate M.2 slot on motherboard
  4. Remove existing drive if present (unscrew standoff screw)
  5. Insert M.2 drive at 30-degree angle
  6. Press down until flat
  7. Secure with screw at end of drive
  8. Reassemble laptop
  9. Boot into BIOS to verify detection
  10. Format/partition in OS

SSD Migration & Data Transfer

Methods to Move Data to New Drive

Fresh Install

  • Install new drive
  • Install clean OS
  • Manually reinstall programs
  • Copy files from backup
  • Pros: Clean, no issues carried over
  • Cons: Time consuming

Cloning

  • Use cloning software (e.g., Macrium, Clonezilla)
  • Creates exact copy of old drive
  • Everything transfers (OS, programs, files)
  • Pros: Fast, everything works
  • Cons: Copies problems too

Storage Best Practices

For Laptops:

  • Use SSD for OS and programs (faster boot, better performance)
  • Keep at least 15-20% free space for optimal performance
  • Never defragment SSDs (wastes write cycles)
  • Enable TRIM for SSDs (usually automatic)
  • Back up important data regularly

For Mobile Devices:

  • Use high-speed microSD cards (A1/A2 class) for apps
  • Format microSD as portable (removable) or internal (Android)
  • Don't remove microSD while device is on
  • Internal storage is always faster than SD card
  • iOS devices cannot expand storage - buy correct size upfront

1. A user wants to upgrade their laptop from an HDD to an SSD for better performance. Which size SSD is a direct replacement for a 2.5" laptop HDD?
Correct! A 2.5" SATA SSD is a direct drop-in replacement for a 2.5" laptop HDD. They use the same SATA interface and physical size, making the swap simple. Just clone or reinstall the OS and you're done.
2. What does the "2280" designation mean for an M.2 SSD?
Correct! M.2 size designations indicate physical dimensions. 2280 means 22mm wide by 80mm long. Other common sizes are 2242 (42mm), 2260 (60mm), and 22110 (110mm). Always check what size your laptop supports.
3. A technician installs an M.2 NVMe drive but it's not detected in BIOS. What should be checked FIRST?
Correct! If BIOS doesn't detect the drive, it's a hardware issue. The most common problem is wrong key type (B vs M vs B+M) or the drive not being properly seated. Check that the physical notches match the slot before anything else.
4. Which mobile device storage type is soldered to the motherboard and cannot be upgraded?
Correct! eMMC (embedded MultiMediaCard) is soldered directly to the device motherboard and cannot be removed or upgraded. It's common in budget smartphones, tablets, and Chromebooks. MicroSD cards are removable and expandable.
5. What is the main difference between M.2 SATA and M.2 NVMe drives?
Correct! Both can be M.2 form factor, but they use different protocols. M.2 SATA uses SATA protocol (~550 MB/s max), while M.2 NVMe uses PCIe lanes (3500+ MB/s). NVMe is significantly faster but more expensive.
6. A user wants to expand storage on their Android phone. What is the BEST option?
Correct! Most Android devices support microSD cards for expandable storage. Internal storage is soldered and cannot be replaced. iPhones/iPads don't support microSD expansion. RAM is memory, not storage.
7. Which microSD card class is recommended for recording 4K video?
Correct! U3 (UHS-I Speed Class 3) or V30 (Video Speed Class 30) both guarantee minimum 30 MB/s write speed, which is required for smooth 4K video recording. Lower classes will result in dropped frames.
8. After installing a new SSD in a laptop, what should NOT be done?
Correct! Never defragment an SSD. It provides no performance benefit (SSDs have instant access) and wastes limited write cycles, reducing the drive's lifespan. Windows automatically disables defragmentation for SSDs.
9. What type of storage is used in flagship smartphones for the fastest performance?
Correct! UFS (Universal Flash Storage) is faster than eMMC and is used in flagship smartphones and high-end tablets. It offers better read/write speeds and lower power consumption than eMMC.
10. A laptop makes clicking noises and runs very slowly. What is the MOST likely issue?
Correct! Clicking noises from a laptop indicate HDD physical failure (read/write head hitting platters). This is the "click of death" and means imminent drive failure. Back up data immediately and replace the drive.
