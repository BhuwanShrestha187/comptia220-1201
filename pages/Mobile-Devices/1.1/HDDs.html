HDD vs SSD Comparison

Feature HDD (Hard Disk Drive) SSD (Solid State Drive)
Technology Spinning magnetic platters Flash memory (no moving parts)
Speed Slower (50-120 MB/s typical) Much faster (200-3500+ MB/s)
Durability Fragile (moving parts) More durable (no moving parts)
Noise Audible spinning/clicking Silent
Power Usage Higher (motor needs power) Lower (better for laptops)
Heat Generates more heat Runs cooler
Price per GB Cheaper More expensive
Capacity Larger (up to 20+ TB) Smaller (common up to 4 TB)
Lifespan 3-5 years average Limited write cycles (5-10 years typical)
Best Use Mass storage, backups OS, applications, performance

HDD (Hard Disk Drive)

How HDDs Work

  • Data stored magnetically on spinning platters
  • Read/write heads move across platters to access data
  • Platters spin at 5400 RPM or 7200 RPM (desktop) or 10,000/15,000 RPM (enterprise)
  • Higher RPM = faster performance but more heat and noise

HDD Form Factors

3.5" HDD

  • Used in desktop computers
  • Larger capacity available
  • Requires more power
  • Common capacities: 500GB - 20TB

2.5" HDD

  • Used in laptops
  • Smaller, less power consumption
  • Lower capacity than 3.5"
  • Common capacities: 250GB - 5TB

HDD Interfaces

Interface Speed Notes
SATA III (6 Gb/s) Up to 600 MB/s Most common, uses L-shaped connector and cable
SATA II (3 Gb/s) Up to 300 MB/s Older standard, backward compatible
PATA/IDE Up to 133 MB/s Legacy, wide ribbon cable, master/slave jumpers
SCSI Varies Enterprise servers, expensive, requires controller card

HDD Failure Signs

  • Clicking/grinding noises: Read/write head failure (imminent failure)
  • Slow performance: Bad sectors developing
  • Files disappearing: Corruption or bad sectors
  • Frequent crashes: Drive failing to read data
  • S.M.A.R.T. errors: Self-monitoring system detecting issues
  • Blue screen errors: Drive communication failures

SSD (Solid State Drive)

How SSDs Work

  • Uses NAND flash memory (same as USB drives)
  • No moving parts = faster, quieter, more durable
  • Data stored in memory cells
  • Limited write cycles per cell (but typically lasts years)
  • Uses wear leveling to distribute writes evenly

SSD Form Factors

2.5" SATA SSD

  • Same size as 2.5" HDD
  • Drop-in replacement for laptop HDDs
  • Uses SATA interface (slower than NVMe)
  • Max speed: ~550 MB/s

M.2 SSD

  • Small stick form factor
  • Plugs directly into motherboard
  • Can use SATA or NVMe protocol
  • Sizes: 2280 (22mm x 80mm) most common

SSD Interfaces/Protocols

Interface Speed Notes
NVMe (PCIe 3.0) Up to 3500 MB/s Fastest, M.2 form factor, direct PCIe connection
NVMe (PCIe 4.0) Up to 7000 MB/s Newer, even faster, requires compatible motherboard
SATA III Up to 600 MB/s Slower than NVMe, 2.5" or M.2 form factor
mSATA Up to 600 MB/s Older, smaller form factor for laptops/tablets
Key Point: M.2 is a form factor (physical size/shape). NVMe and SATA are protocols (how data is transferred). An M.2 drive can use either SATA or NVMe protocol. NVMe is much faster.

Storage for Mobile Devices

Mobile Storage Types

Internal Storage (eMMC/UFS)

  • Soldered to motherboard
  • Cannot be upgraded or replaced
  • Common sizes: 32GB, 64GB, 128GB, 256GB, 512GB, 1TB
  • eMMC: Slower, budget devices
  • UFS: Faster, flagship devices

MicroSD Cards

  • Removable expandable storage
  • Used in Android phones, tablets, cameras
  • Not available on iPhones/iPads
  • Slower than internal storage
  • Can be used to transfer files between devices

Installation & Replacement

Installing 2.5" Drive (HDD/SSD) in Laptop

  1. Power off laptop and remove battery
  2. Remove access panel or bottom cover
  3. Disconnect old drive from SATA connector
  4. Remove mounting screws/bracket
  5. Install new drive in bracket
  6. Connect to SATA connector
  7. Secure and reassemble
  8. Install OS or restore from backup

Installing M.2 SSD

  1. Power off and disconnect power
  2. Locate M.2 slot on motherboard
  3. Remove mounting screw
  4. Insert M.2 drive at 30-degree angle
  5. Press down and secure with screw
  6. Boot and check BIOS recognizes drive
  7. Format and partition in OS
‚ö†Ô∏è Important: M.2 slots have different key notches (B key, M key, B+M key). Make sure the drive matches the slot. Most modern drives are M key or B+M key.

Troubleshooting Storage Issues

Problem Possible Cause Solution
Drive not detected in BIOS Loose cable, dead drive, incorrect jumper Check connections, try different cable/port, check jumpers (PATA)
Clicking/grinding noise (HDD) Physical head failure Backup data immediately, replace drive
Drive detected but no OS Corrupted OS, wrong boot order Check boot order in BIOS, reinstall OS, restore from backup

Data Management

File Systems

File System OS Compatibility Max File Size Notes
NTFS Windows (read/write), macOS (read only) 16 TB Default for Windows, supports encryption
FAT32 All OS (universal) 4 GB Good for USB drives, limited file size
exFAT All modern OS 16 EB Good for large files on removable drives
APFS macOS only 8 EB Optimized for SSDs
ext4 Linux 16 TB Standard Linux file system

Defragmentation

HDDs: Should be defragmented regularly. Files get fragmented across platters, slowing access.

SSDs: NEVER defragment! No benefit (instant access) and wastes limited write cycles.

TRIM (SSD Optimization)

  • TRIM command tells SSD which blocks are no longer needed
  • Helps maintain performance over time
  • Enabled by default in modern operating systems
  • Essential for SSD longevity and speed

1. Which storage type is BEST for an operating system installation when performance is the priority?
Correct! NVMe SSDs are the fastest storage option available, with speeds up to 3500-7000 MB/s compared to HDDs at 50-120 MB/s. For OS installation where boot time and application loading speed matter, NVMe SSD is the best choice.
2. A technician hears clicking and grinding noises from a hard drive. What should be done FIRST?
Correct! Clicking or grinding noises indicate physical head failure in an HDD - the drive is dying. Back up all data immediately before complete failure occurs. This is the "click of death" and means the drive will fail very soon.
3. What is the difference between M.2 and NVMe?
Correct! M.2 is a physical form factor (size and connector type). NVMe is a communication protocol (how data is transferred). An M.2 drive can use either SATA or NVMe protocol. NVMe M.2 drives are much faster than SATA M.2 drives.
4. Which maintenance task should NEVER be performed on an SSD?
Correct! Never defragment an SSD! SSDs have instant access to all data (no moving parts), so defragmentation provides no benefit. It also wastes the SSD's limited write cycles, reducing its lifespan. Windows automatically disables defragmentation for SSDs.
5. What is the maximum theoretical speed of a SATA III interface?
Correct! SATA III (also called SATA 6Gb/s) has a maximum speed of 600 MB/s. SATA II is 300 MB/s. NVMe drives are much faster at 3500-7000 MB/s because they use PCIe lanes instead of the SATA interface.
6. A user wants to expand storage on their Android tablet. What is the BEST solution?
Correct! Most Android tablets support microSD cards for expandable storage. Internal storage is soldered and cannot be replaced. iPhones/iPads do not support microSD expansion. RAM is memory, not storage.
7. Which file system allows files larger than 4 GB and is compatible with all modern operating systems?
Correct! exFAT (Extended FAT) supports files larger than 4 GB and is compatible with Windows, macOS, and Linux. FAT32 has a 4 GB file size limit. NTFS is read-only on macOS. ext4 is Linux-only.
8. A newly installed M.2 NVMe SSD is not appearing in Windows. Where should the technician check FIRST?
Correct! Check BIOS first to verify the drive is physically detected. If the BIOS doesn't see it, there's a hardware issue (loose connection, wrong M.2 slot, incompatible key). If BIOS sees it but Windows doesn't, then check Disk Management to initialize and format it.
9. What is the primary advantage of SSDs over HDDs in laptops?
Correct! SSDs are much faster (200-3500+ MB/s vs 50-120 MB/s) and more durable (no moving parts) than HDDs. This is especially important in laptops that are moved around. HDDs actually have higher capacity and lower cost per GB.
10. Which connection type uses a wide ribbon cable and requires master/slave jumper configuration?
Correct! PATA (also called IDE) uses a wide 40 or 80-wire ribbon cable and requires jumper configuration to set drives as master or slave. This is legacy technology, replaced by SATA which uses thin cables and doesn't require jumpers.

üéØ Key Points for the Exam

  • HDD: Spinning platters, slower (50-120 MB/s), cheaper, fragile, uses for mass storage
  • SSD: Flash memory, faster (200-3500+ MB/s), silent, durable, no moving parts, better for OS/apps
  • SATA III: 600 MB/s max, L-shaped connector, most common interface for 2.5" drives
  • NVMe: Protocol (not form factor), uses PCIe, fastest option (3500-7000 MB/s)
  • M.2: Form factor (physical size), can be SATA or NVMe protocol
  • 3.5" drives: Desktop, 2.5" drives: Laptop/mobile
  • HDD clicking/grinding: Physical failure, backup data IMMEDIATELY
  • Never defragment SSDs: Wastes write cycles, no benefit
  • TRIM: SSD optimization command, keeps performance high
  • MicroSD cards: Expandable storage for Android (not iOS), check speed class for use
  • PATA/IDE: Legacy, wide ribbon cable, master/slave jumpers required
  • File systems: NTFS (Windows), FAT32 (universal, 4GB limit), exFAT (universal, large files)
  • M.2 keys: B, M, or B+M - drive must match slot notch
  • S.M.A.R.T.: Self-Monitoring system predicts drive failure
  • Bluetooth keyboard won't work in BIOS: Use wired USB for BIOS access
