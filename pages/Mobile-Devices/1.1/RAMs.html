Monitoring RAM & Common Failure Symptoms

CompTIA tests recognition of RAM-related issues in mobile devices:

  • Frequent system crashes / freezes / Blue Screen of Death (BSOD) with codes like MEMORY_MANAGEMENT
  • Random reboots or unexpected shutdowns
  • Sluggish performance / high memory usage even with few apps
  • Application crashes or corrupted data/files
  • POST beeps (memory error codes on some laptops)
  • Failure to boot (no display or stuck at logo)
MEMORY_MANAGEMENT BSOD â€” one of the classic signs of bad or failing RAM (very common on exams).

Monitoring tools:

  • Windows Memory Diagnostic (mdsched.exe)
  • Task Manager / Resource Monitor (high usage / errors)
  • Third-party: MemTest86 (bootable USB for thorough testing)

Q1. What is the standard RAM form factor used in modern laptops?
Correct Answer: B) SODIMM
Explanation: Laptops use Small Outline Dual In-line Memory Module (SODIMM). Desktop computers use full-size DIMM. DDR4/DDR5 refer to generations, not the form factor.
Q2. A laptop displays frequent MEMORY_MANAGEMENT BSOD errors. What component is most likely faulty?
Correct Answer: C) RAM
Explanation: MEMORY_MANAGEMENT is a classic Windows Blue Screen error directly related to memory (RAM) corruption or failure.
Q3. What is the FIRST safety step before replacing RAM in a laptop?
Correct Answer: A) Power off the device, unplug the charger, remove the battery (if removable), and use ESD protection
Explanation: Eliminate all power sources and prevent electrostatic discharge (ESD) before opening the laptop.
Q4. At what angle should a SODIMM module be inserted into the slot during installation?
Correct Answer: B) 45 degrees
Explanation: SODIMMs are inserted at approximately 45Â° and then pressed down until the side retaining clips snap into place.
Q5. A technician installs new RAM, but the laptop does not POST. What is the MOST likely cause?
Correct Answer: D) Incompatible RAM type, speed, or generation
Explanation: DDR4 and DDR5 modules are physically and electrically incompatible (different pin count, notch position, voltage). Mismatch prevents the system from booting.
Q6. Which built-in Windows tool is used to test for RAM errors?
Correct Answer: A) Windows Memory Diagnostic
Explanation: Run mdsched.exe (Windows Memory Diagnostic) to perform extended memory tests. MemTest86 is third-party but also excellent.
Q7. Why is ESD protection mandatory when handling laptop RAM?
Correct Answer: C) Static electricity can instantly damage sensitive RAM chips
Explanation: RAM modules are extremely sensitive to electrostatic discharge. Always use an anti-static wrist strap or touch a grounded metal surface.
Q8. Which TWO of the following are common symptoms of failing laptop RAM? (select 2)
Correct Answer: A) Frequent BSODs and C) Random system reboots
Explanation: These are direct indicators of memory instability/corruption. Sluggish performance can have many causes; corrupted files are more often storage-related.
